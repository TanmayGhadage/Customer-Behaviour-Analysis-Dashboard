# Customer Shopping Behavior Analysis Dashboard

## Overview

This project analyzes customer shopping behavior using transactional data to identify purchasing patterns, customer segments, product preferences, and revenue trends. The analysis was performed using Python, SQL, PostgreSQL, and Power BI to generate actionable business insights and interactive visualizations.

---

## Project Objectives

* Analyze customer purchasing behavior and spending patterns.
* Identify high-value and loyal customers.
* Compare subscriber and non-subscriber performance.
* Evaluate the impact of discounts on customer purchases.
* Discover top-performing products and categories.
* Build an interactive dashboard for business decision-making.

## Dataset Information

The dataset contains customer transaction records including:

* Customer Demographics
* Product Categories
* Purchase Amount
* Review Ratings
* Subscription Status
* Shipping Preferences
* Discount Usage
* Purchase Frequency

Total Records: 3,900+

---

## Data Cleaning & Preprocessing

* Handled missing values.
* Standardized column names.
* Performed data validation checks.
* Created age group categories.
* Generated purchase frequency metrics.
* Loaded cleaned data into PostgreSQL for analysis.

## SQL Business Analysis

The following business questions were answered using SQL:

1. Revenue generated by Gender.
2. High-spending customers using discounts.
3. Top-rated products.
4. Shipping type comparison.
5. Subscribers vs Non-Subscribers analysis.
6. Discount-dependent products.
7. Customer segmentation.
8. Top products by category.

10. Revenue contribution by age group.

## Key Insights

* Loyal customers contribute a significant portion of total revenue.
* Subscribers tend to spend more than non-subscribers.
* Certain products show strong dependency on discount campaigns.
* Customer purchase behavior varies across age groups.
* Product ratings influence purchasing decisions.
